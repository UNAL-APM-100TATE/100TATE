Esta sección tiene como objetivo analizar los elementos involucrados un proceso de manufactura y asociarlos a indicadores de producción,además de utilizar indicadores 
de manufactura para validar la eficiencia y eficacia de un proceso aplicando tecnologías de automatización.En nuestro proyecto de tuvieron en cuenta los distintos costos,
tales como la adquisición de la materia prima, la maquinaria requerida, salarios, arriendo, publicidad, servicios públicos, software y obligaciones bancarias. Con 
estos costos, a continuación, se realizó una solicitud de crédito para la adquisición de maquinaria y con este valor se halló diferentes indicadores financieros como 
VAN, TIR e IR.
